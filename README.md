# Customer Churn Prediction

Customer churn is one of the most critical problems for subscription-based businesses. If companies can predict which customers are likely to leave, they can take preventive measures and improve customer retention.

This project builds a machine learning model to predict whether a customer will churn based on historical customer data.

---

## 📌 About the Project

The objective of this project is to:

- Analyze customer demographic and service usage data
- Perform data preprocessing and feature engineering
- Train machine learning models
- Evaluate model performance
- Identify key factors influencing churn

This project covers the complete machine learning pipeline from data cleaning to model evaluation.

## 🔍 Key Insights

- Customers with shorter tenure are more likely to churn.
- Monthly charges have a strong impact on churn probability.
- Customers on long-term contracts tend to stay longer.
- Certain service features influence customer retention.
